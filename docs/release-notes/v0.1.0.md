## plexget v0.1.0

First release — an SFTP-style terminal browser and downloader for your Plex media.

### Highlights
- **PIN login** via `plex.tv/link` (2FA-friendly), token cached for next time.
- **Arrow-key TUI** to browse TV and Movie libraries with type-to-filter.
- **Enter** downloads a single episode/movie; **→** downloads a whole
  show/season/movie folder after a "N files, X GB?" confirmation.
- **Original-file** streaming with a live **speed (MB/s) + ETA** progress bar.
- **`--segments N`** parallel-range mode for faster single-file downloads.
- Mirrors the `Show/Season/…` layout on whole-folder downloads, skips
  already-downloaded files, cleans up partial files on interrupt.

### Install
Download the binary for your platform from the assets below and run it — no
Python required. On macOS you may need to clear the quarantine attribute
(`xattr -d com.apple.quarantine ./plexget-macos-arm64`). See the
[README](https://github.com/keithah/plexget#install) for details.
